# 酒水商城 (Drink Mall)

## What This Is

酒水商城是一个微信小程序电商平台，专注于酒水商品的在线零售。用户可以浏览商品、下单购买、管理订单，平台运营方可通过后台管理系统进行商品、订单、用户、内容的全面管理。

**注意：本项目已剔除所有分销相关功能（联营商/合伙人等级体系、分销奖励、招商专区、团队业绩、分红、DF余额、上级绑定等），仅保留核心电商功能。**

## Core Value

用户可以方便地在微信小程序中浏览和购买酒水商品，享受流畅的购物体验，平台可以高效管理商品和订单。

## Context

- 微信小程序电商平台
- 酒水商品零售场景
- 需支持1000并发用户
- 所有商品包邮，无配送费
- 积分兑换体系（礼包专区）
- 微信支付对接

## Constraints

- **平台**：微信小程序 + PC后台管理端
- **支付**：对接微信支付
- **物流**：对接物流查询API
- **并发**：支持1000人在线
- **性能**：小程序首屏≤2秒，后台≤1.5秒
- **安全**：符合微信小程序规范及网络安全法规
